# Projeto Calculadora SymPy 
Este projeto consiste em uma calculadora de expressões algébrica desenvolvida em linguagem Python, com suporte da biblioteca SymPy para a manipulação de matemática.

# Funcionalidades
* Manipulação algébrica e fatoração automatizada de expressões matemáticas complexas.
* Substituição dinâmica de variável para resolução e obtenção de resultados numéricos otimizados.
* Sistema robusto de tratamento de exceções para validação de dados de entrada, garantindo a estabilidade da aplicação contra falhas de digitação.

# Modo
* Python
* SymPy
